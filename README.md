# Honda Price Predictor

In this project the data on Honda vehicles listed on kijiji is obtained through scraping, cleaned and transformed and eventually stored in SQL. 
Then the data is read from SQL and some analysis and visualization is performed. Also different Machine Learning Models are used to perdict the 
price of a used Honda vehicle on the basis of diffrenet features such as its model, year, transmission type and milage.
